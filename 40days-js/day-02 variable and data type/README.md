## 🧠 JavaScript Data Types

### 🔹 Primitive Data Types

- String → "Hello"
- Number → 25, 39
- Boolean → true / false
- Undefined → variable declared but not assigned
- Null → empty value
- Symbol → unique identifier
- BigInt → large numbers
